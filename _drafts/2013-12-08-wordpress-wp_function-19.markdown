<p># dynamic_sidebar<br />
- Codex reference -- http://codex.wordpress.org/Function_Reference/dynamic_sidebar<br />
- This function calls each of the active widget callbacks in order, which prints the markup for the sidebar.<br />
- If you have more than one sidebar, you should give this function the name or number of the sidebar you want to print.<br />
- This function returns true on success and false on failure.<br />
- The return value should be used to determine whether to display a static sidebar. This ensures that your theme will look good even when the Widgets plug-in is not active.</p>
